Built for real-world RTL

Three powerful capabilities that just work.

Smart Auto Mode

Detects Hebrew, Arabic, and Persian characters in real-time. Only applies RTL direction to elements that actually contain RTL text — leaving everything else untouched.

Per-Site Persistence

Set it once, forget it forever. Your preferred mode is saved per website and synced across all your Chrome instances via cloud storage.

Zero Overhead

No dependencies. No performance hit. Built with pure browser APIs and a MutationObserver that activates only when needed. Your pages stay fast.

Three modes, total control

Choose how BiDi handles each website.

NONE

Off

Standard browser behavior. BiDi won't touch the page.

AUTO
Recommended

Auto Detect

Scans each element for RTL characters and applies direction only where needed. Handles mixed content perfectly.

RTL

Full RTL

Forces right-to-left on the entire page. Best for sites that are fully in Hebrew, Arabic, or Persian.

Get started in 30 seconds

Available as a developer extension while we await Chrome Web Store approval.

1

Clone the repository

git clone https://github.com/ekrako/bidi.git
2

Build the extension

bun install && bun run build
3

Load in Chrome

Go to chrome://extensions, enable Developer Mode, click "Load unpacked" and select the dist/ folder.
