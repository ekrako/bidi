Smart Auto Mode

Automatically detects RTL characters (Hebrew, Arabic, etc.) and applies direction only to elements containing RTL text.

Per-Site Persistence

Your settings are saved per-hostname and synced across all your Chrome instances using chrome.storage.sync.

Fast & Lightweight

How to Install

BiDi is currently in development. You can load it as a developer extension:

  1. Clone the repository.
  2. Run bun install and bun run build.
  3. Go to chrome://extensions/ in Chrome.
  4. Enable Developer mode.
  5. Click Load unpacked and select the dist/ folder.
